# DeepSeek V3.2

DeepSeek V3.2 is a powerful, open-source Mixture-of-Experts (MoE) language model, boasting an impressive 671 billion total parameters.

## Why DeepSeek V3.2?

- **Narrative Detail:** Its vast parameter count allows for a high degree of imaginative detail and logical consistency in complex world-building.
- **Efficient Context:** Features a **128K context window**, providing stable and reliable performance for chapter-length drafts and detailed scene outlines.

## Rate Limits & Access

DeepSeek V3.2 is available via Ollama's cloud infrastructure, ensuring high-performance access without local hardware constraints.

> [!NOTE]
> **Cloud Rate Limits**
> To maintain capacity for the writing community, Ollama implements **hourly and weekly free usage rates**. These limits allow for extensive brainstorming sessions while preventing system congestion.

## How to Get Your API Key

To access DeepSeek V3.2 in SammyAI, you must use your Ollama API key:

1. **Sign Up / Sign In:** Visit [ollama.com](https://ollama.com/) and ensure you are logged in.
2. **Retrieve API Key:** Access your account dashboard to find or generate your cloud API key.
3. **Configure SammyAI:** Copy your key and paste it into the SammyAI API settings (Settings > API Keys) under the **Ollama** provider.
